# Linux

## Requirements:
- Docker
- Docker-Compose
- Your Linux User UID is 1000 (Run `id -u`)

## Installation
* Clone the repository somewhere
* Do a symlink from swdc to `/usr/local/bin/swdc`
* Run once swdc to generate configuration
* Run `swdc up` to start all containers
